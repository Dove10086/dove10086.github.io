Python中的一些算法表达式

1. ‘/‘号

语法
```Python
1.3 / 2
```
含义3除以2,结果1.5

2. ‘//‘号

```Python
语法
3 // 2
```
含义
3除2取小于被除数的最大整数（也叫地板除），

Python中的一些内置函数(BIF)

divmod(A,B)

同时求出两(A,B)地板除结果和余数的值
语法:
Python divmod(3,2) 答案（1,1）

abs(x)

x的绝对值

int(x)

含义将x转换为整数

float(3.14)

将3.14转换成浮点型

complex(re,im)

返回一个复数,re是实部，im是虚部

pow(x,y)

计算x的y次方

x ** y

计算x的y次幂
