语句用法

目标当变量A大于10时输出你好反之输出你好我们不熟

A = input("请输入一个数字")
A = int(A)

if A >= 10:
    print("你好")
else:
    print("你好我们不熟")

三元表达式

在python中有一个比较奇特的写法三元表达式
语法

变量 = 值1 if 条件 else 值2

A = input("请输入一个数字")
A = int(A)

print("你好") if A >= 10 else print("你好我们不熟")

在比较复杂的循环结构中我更推荐前者这样更直观更容易修改
